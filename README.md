# اسکنر اختیار معامله — ربات تلگرام

## مراحل راه‌اندازی

### قدم ۱: ساخت ربات تلگرام
1. در تلگرام به @BotFather پیام بدید
2. بنویسید /newbot
3. یک نام انتخاب کنید
4. TOKEN دریافت‌شده را کپی کنید

### قدم ۲: گرفتن Chat ID
1. ربات را در تلگرام باز کنید
2. /start بزنید
3. Chat ID نمایش داده می‌شود — آن را کپی کنید

### قدم ۳: Deploy روی Railway (رایگان)
1. به railway.app بروید و ثبت‌نام کنید
2. New Project → Deploy from GitHub Repo
3. این فایل‌ها را آپلود کنید
4. در بخش Variables این متغیرها را اضافه کنید:
   - BOT_TOKEN = توکن ربات شما
   - CHAT_ID = شناسه چت شما
   - RISK_FREE_RATE = 0.25
   - DISCOUNT_THRESH = 0.05
   - IV_PERCENTILE = 20
   - SCAN_INTERVAL_MIN = 30

### دستورات ربات
- /scan — اسکن فوری
- /status — وضعیت
- /help — راهنما

### نکات
- ربات فقط در ساعت بازار (۹:۰۰ تا ۱۲:۳۰) اسکن خودکار می‌کند
- با /scan هر زمانی می‌توانید اسکن فوری بگیرید
- اگر فیلترشکن روشن باشد داده واقعی از TSETMC می‌گیرد
- بدون فیلترشکن از داده شبیه‌سازی استفاده می‌کند
